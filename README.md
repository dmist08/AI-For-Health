# AI for Health — SRIP 2026 Project
This repository contains the complete implementation for the SRIP 2026 Health Sensing assignment, focusing on detecting breathing irregularities (Apnea and Hypopnea) during sleep using Multi-Branch 1D Convolutional Neural Networks.

## Running the Pipeline (Google Colab Recommended)

Given the computational scale of LOPO Cross-Validation across millions of parameters, we recommend executing the end-to-end pipeline in Google Colab. The steps are fully automated:

1.  **Open `notebooks/AI_for_Health.ipynb` in Google Colab**.
2.  Upload the raw `internship.zip` containing the assignment data.
3.  **Run All Cells**: The notebook seamlessly orchestrates data structuring (`setup_data.py`), dataset extraction (`create_dataset.py`), multi-mode training execution, and metric table compilation.

### Key Technical Contributions
*   **No Data Leakage:** Z-score normalization is strictly calculated per-fold dynamically at runtime, applying training normalization constants exactly to the hold-out test set.
*   **Multi-Mode Evaluation:** Provides separate event models to isolate Apnea distribution skews.
*   **Robust Parsers:** Can handle any permutations of file variations robustly via Regex standardizations.
