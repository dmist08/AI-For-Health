Health Sensing [15 Marks]


Your team is working on detecting breathing irregularities that occur during sleep. As part of a pilot study,  your team has collected overnight sleep data (8 hours per participant) from 5 subjects. The firm has provided you with the below Dataset. 


For each participant, the following physiological signals are available:
* Nasal Airflow (sampled at 32 Hz)
* Thoracic Movement (sampled at 32 Hz)
* SpO₂ (Oxygen Saturation) (sampled at 4 Hz)

In addition to these signals, the dataset also includes:
   * An event file that contains annotations for breathing irregularities (e.g., apnea, hypopnea).
   * A sleep profile file that records sleep stages over time.


Your task is to analyze and model this data to detect abnormal breathing patterns during sleep.
Understanding the Data and Visualization [4 Marks]
Your manager would like to explore how the recorded signals look across the 8-hour sleep sessions. Your task is to create clear visualizations of the data for each participant, making it easier to interpret patterns and irregularities.
   * Plot Nasal Airflow, Thoracic Movement, and SpO₂ for the full 8-hour recording of one participant. (2 marks)
   * Overlay annotated breathing events on the plots. (1 mark)
   * Save the visualization as a PDF file. (1 mark)
Hints
   * The respiration signals and SpO₂ signals have different sampling rates (32 Hz vs 4 Hz).
   * Fortunately, all signals include timestamps. You can treat them as time series and align them accordingly.
   * Consider using Pandas for handling and aligning the time-indexed data. Making good use of timestamps will be essential to synchronize and plot the signals accurately.
Deliverable
Write a script vis.py that takes a participant folder as input and generates the PDF.
Example usage:
python vis.py -name "Data/AP20"
Here, AP20 is the folder containing the signal files for one participant. The script should generate a PDF visualization and store it in a directory called Visualizations.
Signal Preprocessing and Dataset Creation [6 Marks]
Human breathing typically occurs at a rate of 10 to 24 breaths per minute (BrPM), which corresponds to a frequency range of approximately 0.17 Hz to 0.4 Hz. Signals containing frequency components significantly higher than this range are likely noise.


   * Apply digital filtering techniques to retain only the relevant breathing frequency range. Consider using libraries such as SciPy, NumPy, or PyWavelets to implement your filtering pipeline( 2 marks) 
   * Split the signals into 30-second windows with 50% overlap.(2 marks)
   * Use the flow events file to determine the label for each window:(2 marks) 
   * If a window overlaps by more than 50% with a labeled event (e.g., Hypopnea or Obstructive Apnea), assign that event’s label to the window.
   * If there is no sufficient overlap with any event, label the window as Normal.
Save the processed dataset in a suitable format (e.g., CSV or Pickle). 
Deliverable
Write a Python script named create_dataset.py that:
   * Reads the input signals and annotations from the Data directory.
   * Processes the data into labeled windows.
   * Saves the resulting dataset to the Dataset directory.
Example usage
python create_dataset.py -in_dir "Data" -out_dir "Dataset"
Basic Modeling and Evaluation [5 Marks]
Your objective is to train models to classify breathing irregularities based on the labeled 30-second windows.
Modelling (2 marks) 
   * Train a simple 1D CNN model for classifying windows. 
Evaluation Strategy (1 mark)
You must evaluate your models using Leave-One-Participant-Out Cross-Validation. 
   * In each fold, train on the data from 4 participants and test on the remaining one and repeat this process for all 5 participants.


Metrics to Report (2 marks)
   * Report Accuracy, Precision, Recall, and Confusion Matrix. (2 marks)
